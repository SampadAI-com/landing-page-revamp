# SampadAI Revamp Web

A modern Next.js landing page with 6 sections and a waitlist feature.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Sections

1. **Hero** - Main landing section with call-to-action buttons
2. **Features** - Three key features with icons
3. **About** - Information about SampadAI with statistics
4. **Services** - Four service offerings
5. **Testimonials** - User testimonials
6. **Waitlist** - Email signup form for early access
7. **Contact** - Contact information and message form

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
